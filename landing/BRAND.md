# BRAND.md — mcpbolt-landing
> What every agent must know before touching this project.

## Product

- **Name:** MCPBolt (mcpbolt)
- **Domain:** mcpbolt.app
- **One sentence:** MCPBolt installs, syncs, and manages MCP servers across Claude Desktop, Cursor, VS Code, and 10 more AI coding tools — from one menu bar icon.
- **Tagline:** "One-click MCP servers for every app"
- **CLI:** `npx mcpbolt` — free, MIT open source
- **Mac app:** MCPBoltBar — $29 lifetime, closed source

## User

- **Who it's for:** Developers (especially vibe coders) using 2+ AI coding tools who want MCP servers working everywhere without editing JSON config files
- **Problem it solves:** Every AI coding tool stores MCP configs differently. mcpbolt translates and installs in one step.
- **What it is NOT:** A team tool, a cloud service, a chat app, an IDE, or a subscription product

## Personality

- **Tone:** Fast and opinionated. Confident. Speaks to developers who know their tools.
- **Values:** Local-only, privacy-first, atomic writes, timestamped backups, no cloud, no account
- **Messaging pillars:**
  1. One menubar icon. Every AI coding tool. Every hidden setting.
  2. 12 editors supported — twice as many as the next tool.
  3. Local-only, atomic writes, timestamped backups, no cloud.
